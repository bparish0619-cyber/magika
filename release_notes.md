## Magika Android App - Initial Release 🚀

We are excited to provide the first release of the **Magika Android Application**! This app wraps the powerful AI-driven Magika Content-Type Detection model into a user-friendly, minimalistic mobile interface.

### ✨ Features
- **Sleek Dark Mode UI:** A clean, glassmorphism-inspired design layout focused on usability.
- **Native Android File/Folder Picker:** Deep integration with Android's built-in file picker using Scoped Storage.
- **On-Device ML Inference:** Files are read securely and analyzed fully offline using the `magika` ML engine.
- **Detailed Detections:** Tap on any detected file to view detailed metadata, including the full system path, text-classification status, exact confidence scores, and more!
- **Selective Result Clearing:** Easily remove individual scans or clear out your entire session.

### 📥 Installation Instructions
1. Download the attached `magika-android-v1.0.apk` under **Assets** below.
2. Transfer the file to your Android device.
3. Open the file on your device to install. *(You may need to enable "Install Unknown Apps" from your browser or file manager).*

Enjoy lightning-fast AI file-type detection on the go!
